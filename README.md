
# Toy Duration Predictor Model

preprocess SVS dataset with [midii](https://github.com/ccss17/midii) 

## Installation

```shell
pip install git+https://github.com/ccss17/toy-duration-predictor.git
```

## Usage

```python
import toy_duration_predictor as tdp
```
